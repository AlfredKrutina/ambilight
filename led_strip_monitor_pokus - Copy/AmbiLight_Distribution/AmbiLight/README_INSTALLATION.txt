========================================================================
             AmbiLight - Installation Instructions
========================================================================

Thank you for downloading AmbiLight!

System Requirements:
- Windows 10 or 11
- Microsoft Visual C++ 2015-2022 Redistributable (x64)
  (If the app doesn't start, download it from Microsoft: https://aka.ms/vs/17/release/vc_redist.x64.exe)

------------------------------------------------------------------------
STEP 1: INSTALLATION
------------------------------------------------------------------------
1. Move this "AmbiLight" folder to a permanent location on your PC.
   Recommended locations:
   - C:\Program Files\AmbiLight
   - C:\Users\<YourUser>\Documents\AmbiLight

   IMPORTANT: Do NOT run it directly from the ZIP file or a temp folder.

2. Double-click the "INSTALL_WINDOWS.bat" file.
   - This script will automatically:
     a) Create an "AmbiLight" shortcut on your Desktop.
     b) Add AmbiLight to your Windows Startup folder so it starts when you log in.

------------------------------------------------------------------------
STEP 2: RUNNING THE APP
------------------------------------------------------------------------
- You can launch the app using the Desktop shortcut you just created.
- Or double-click "AmbiLight.exe" directly.

- The app runs in the background. Look for the colorful icon in your
  System Tray (near the clock).
- Right-click the tray icon to access:
  - Settings
  - Calibration
  - Effects
  - Exit

------------------------------------------------------------------------
TROUBLESHOOTING
------------------------------------------------------------------------
- If the app fails to start, try running it as Administrator once.
- Ensure your antivirus isn't blocking "AmbiLight.exe".
- Check if you have the Visual C++ Redistributable installed (link above).

Enjoy!
