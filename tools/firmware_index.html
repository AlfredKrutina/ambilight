Desktop — Windows a macOS

Načítám poslední release…

  • Všechny releases
  • desktop-manifest.json (auto-update / kontrola verze)
Windows: doporučujeme Setup (.exe) — klasická instalace do Program Files. ZIP je portable složka (vhodné pro auto-update v aplikaci).
macOS: DMG — aplikace z CI není notarizovaná; při prvním spuštění může být potřeba Systémová nastavení → Soukromí a zabezpečení nebo pravý klik → Otevřít.

Firmware (ESP32-C3 lampa)

Nejnovější build CI: firmware/latest/manifest.json a soubory .bin ve stejné složce.

V aplikaci: Nastavení → Firmware — stažení, USB flash (esptool), nebo OTA přes UDP / MQTT.
