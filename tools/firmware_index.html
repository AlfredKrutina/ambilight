Podívej se zblízka.

Čtyři vizuály na stránce: hlavní obrázek v úvodu (hero-poster.png) + tři náhledy níže — stejný hero motiv, fauna a sport. Světlo za panelem odpovídá jen okrajům obrazu. Screenshoty z reálné aplikace můžeš doplnit pod galerií — viz tools/pages_assets/README.md.

Hero — pobřežní louka, zlatá hodina; halo výhradně z okrajových pruhů obrazovky.

Fauna — skandinávský prostor, úhel zleva; spill podle levého a pravého okraje.

Sport — loft, záběr shora zprava; kontrast vlevo–vpravo jen z okrajů obrazovky.

Stažení.

Nejnovější desktopové buildy odpovídají tagu desktop-v* (workflow desktop_release). Primární odkazy vedou na github.com — přímé stažení z release; volitelná kopie je i na této doméně v desktop/latest/ po deployi Pages.

Načítám release z GitHub API…

  • Všechny releases

Manifest a zrcadlený tag
  • desktop-manifest.json — verze / auto-update (GitHub release)
  • desktop-release-tag.txt — jen mirror na Pages
Windows: Setup → Program Files; ZIP → portable / auto-update. macOS: při blokaci systémová nastavení nebo pravý klik → Otevřít. Linux: např. apt install libgtk-3-0 libayatana-appindicator3-1.

Firmware.

ESP32-C3 lampa — nejnovější CI build ve složce firmware/latest/. V aplikaci: Nastavení → Firmware.
