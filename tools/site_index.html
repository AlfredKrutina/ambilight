Open source · ESP32 · Flutter desktop

AmbiLight

Screen-aware ambient lighting with a cross-platform desktop client and lamp firmware. This site hosts CI-built packages (Windows & Linux) plus the latest ESP-IDF binaries for OTA/USB workflows.

Desktop application

Builds are produced by GitHub Actions from the default branch. Extract the archive and run the app bundle inside. macOS is not published here (signing & notarization); build locally with Flutter — see the repository runbook.
